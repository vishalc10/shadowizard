# What is this?

Get perfect shadows every time for the non-designer.

# Installation

`npm i vshadowizard --save`

Then...

```
import { vshadowizard } from 'vshadowizard';

vshadowizard({
    shadow_type: 'soft',
    padding: false,
});
```

## Options

- _shadow_type_ - _hard / soft_ (Defaults to soft)
- _padding_ - _boolean_ (Defaults to flase)
